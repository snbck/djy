<tr><td VALIGN=TOP width="626"><h2 align=center>归功妈妈从小叮咛 袁咏琳养成环保习惯</h2>
<img width="600" src="http://i.epochtimes.com/assets/uploads/2020/01/20200116-Cindy-01-2-600x400.jpg" />
<h6>Cindy袁咏琳受邀担任“2020 Nat Geo Run 国家地理路跑”环保大使。（国家地理提供）
</h6>
<hr>
<p>【大纪元2020年01月16日讯】（大纪元记者杨逸帆报导）Cindy<a href="https://github.com/zsifxm298/djy/blob/master/gb/tag/%E8%A2%81%E5%92%8F%E7%90%B3.md">袁咏琳</a>最近受邀担任“2020 Nat Geo Run <a href="https://github.com/zsifxm298/djy/blob/master/gb/tag/%E5%9B%BD%E5%AE%B6%E5%9C%B0%E7%90%86%E8%B7%AF%E8%B7%91.md">国家地理路跑</a>”的<a href="https://github.com/zsifxm298/djy/blob/master/gb/tag/%E7%8E%AF%E4%BF%9D.md">环保</a>大使，具有歌手、演员、外景主持多种身份的她，谈到自己环保习惯，Cindy立刻归功妈妈，从小不停地在各种生活细节上叮咛她，经常做久而久之就养成的好习惯。</p>
<p>从小Cindy妈妈就时常告诫不要浪费水电，妈妈只要听到单纯地流水声，关水龙头的叮咛声就会传来，妈妈经过房间看到还不到开灯时间，也会进来随手关掉，甚至只是离开房间一下下，妈妈也会立刻帮她把灯关掉。她说：“但现在想起来，从小若没有妈妈这样逼着我做，可能也不容易养成随手关电关水的习惯。”</p>
<figure id="attachment_11798529" style="width: 600px" class="wp-caption aligncenter"><a href="http://i.epochtimes.com/assets/uploads/2020/01/20200116-Cindy-02.jpg"><img src="http://i.epochtimes.com/assets/uploads/2020/01/20200116-Cindy-02-600x899.jpg" alt="Cindy 袁咏琳" width="450" b="899" class="size-large wp-image-11798529" /></a><figcaption class="wp-caption-text">Cindy<a href="https://github.com/zsifxm298/djy/blob/master/gb/tag/%E8%A2%81%E5%92%8F%E7%90%B3.md">袁咏琳</a>受邀担任“2020 Nat Geo Run <a href="https://github.com/zsifxm298/djy/blob/master/gb/tag/%E5%9B%BD%E5%AE%B6%E5%9C%B0%E7%90%86%E8%B7%AF%E8%B7%91.md">国家地理路跑</a>”<a href="https://github.com/zsifxm298/djy/blob/master/gb/tag/%E7%8E%AF%E4%BF%9D.md">环保</a>大使。（国家地理提供）</figcaption></figure>
<p>另一个好习惯，是她对学习新事物的毅力坚持，设定好目标非做到不可。像学跳舞就是一例，现在则是热衷于网球运动；Cindy表示：“网球一打就是2、3个小时，可以加强自己的爆发力与肌耐力，这些都有助于我在舞台上或是MV的演出。”</p>
<p>而跑步则是喜欢找爱跑步的同好一起练习，Cindy表示：“因为跑一跑觉得快要不行想放弃时，可以互相激励告诉对方不要停，要继续跑完！”</p>
<p>这次2020 Nat Geo Run国家地理<a href="https://github.com/zsifxm298/djy/blob/master/gb/tag/%E8%B7%AF%E8%B7%91.md">路跑</a>，Cindy希望能超越自己挑战9公里，但考量当天若还有演出，可能会选择参加4公里组。 </p>
<figure id="attachment_11798532" style="width: 600px" class="wp-caption aligncenter"><a href="http://i.epochtimes.com/assets/uploads/2020/01/20200116-Cindy-03.jpg"><img src="http://i.epochtimes.com/assets/uploads/2020/01/20200116-Cindy-03-600x899.jpg" alt="Cindy 袁咏琳" width="450" b="899" class="size-large wp-image-11798532" /></a><figcaption class="wp-caption-text">Cindy袁咏琳受邀担任“2020 Nat Geo Run 国家地理<a href="https://github.com/zsifxm298/djy/blob/master/gb/tag/%E8%B7%AF%E8%B7%91.md">路跑</a>”环保大使。（国家地理提供）</figcaption></figure>
