<p>【大纪元2019年11月15日讯】大纪元每天为读者梳理<a href="https://github.com/fwwipt2046/djy/blob/master/gb/tag/%E7%BF%BB%E5%A2%99%E5%BF%85%E7%9C%8B.md">翻墙必看</a>的文章：</p>
<p>1 . <a href="https://github.com/fwwipt2046/djy/blob/master/gb/19/11/14/n11655964.md">【新闻看点】中共露杀机香港危急 美警告有代价</a><br />
正在巴西参加金砖国家会议的习近平表示，香港持续发生的激进暴力犯罪行为，严重践踏法治和社会秩序。“止暴制乱、恢复秩序是香港当前最紧迫的任务。”这是习近平在反送中五个多月中，第一次明确对香港局势作出公开评论。外界担心，中共已经失去了耐性，显露杀机。</p>
<p>2. <a href="https://github.com/fwwipt2046/djy/blob/master/gb/19/11/14/n11655449.md">香港危急 大规模血腥镇压露先兆</a><br />
本周以来，香港局势急剧恶化。<a href="https://github.com/fwwipt2046/djy/blob/master/gb/tag/%E6%B8%AF%E8%AD%A6.md">港警</a>疯狂抓捕、催泪弹扫射示威者，甚至向示威者开枪，并闯入香港中文大学，向学生发射大量催泪弹及橡胶子弹，令校园宛如战场。一幕幕血腥画面震惊国际社会。然而，从各种迹象及中共惯用手法显示，香港更大规模的血腥镇压先兆已经出现。</p>
<p>3. <a href="https://github.com/fwwipt2046/djy/blob/master/gb/19/11/14/n11656487.md">中大保卫战 学生命在旦夕 吁国际人道救援</a><br />
由于<a href="https://github.com/fwwipt2046/djy/blob/master/gb/tag/%E6%B8%AF%E8%AD%A6.md">港警</a>强行攻入大学校园而引发的一场“<a href="https://github.com/fwwipt2046/djy/blob/master/gb/tag/%E4%B8%AD%E5%A4%A7%E4%BF%9D%E5%8D%AB%E6%88%98.md">中大保卫战</a>”，14日步入第四天，局势依然牵动人心。中文大学变战场，尽管中共已急令内地学生撤回大陆，台湾侨生撤走，美国、印尼等国也撤走其交换生，仍有数千名学生、教职员等彻夜互助守护校园，场面感人。学生命运危在旦夕，急需国际救援。</p>
<p>4. <a href="https://github.com/fwwipt2046/djy/blob/master/gb/19/11/14/n11655927.md">香港局势升级 习近平发话 强硬表态</a><br />
香港局势进一步升级，中共领导人习近平14日针对香港问题公开表态，并发表强硬讲话。美国国会迅速出手，就《香港人权民主法案》启动“热线”机制。</p>
<p>5. <a href="https://github.com/fwwipt2046/djy/blob/master/gb/19/11/14/n11654353.md">【十字路口】七大讯号 中共或重演六四镇压</a><br />
目前香港反送中运动的局势依然越来越紧张，不仅仅是因为在中共领导人公开支持林郑月娥后，香港警方的暴力越来越升级，而在过去这几天以来，也陆续出现了多项讯号，暗示着中共当局有可能准备发动大规模的武力镇压，对反送中运动进行全面清场。不管是穿着军服还是穿着警服，都等于是发起一场“香港版六四事件”。</p>
<p>6. <a href="https://github.com/fwwipt2046/djy/blob/master/gb/19/11/14/n11656097.md">美参院启动“热线” 加速表决香港人权法案</a><br />
美国参议院多数党领袖麦康奈尔11月14日同意对《香港人权与民主法案》启动“热线”（hotline）机制。此举意味着该法案将进入快速通道。目前已经有40位两党参议员签署此法案。</p>
<p>7. <a href="https://github.com/fwwipt2046/djy/blob/master/gb/19/11/14/n11655579.md">英国政府将出台法律 制裁林郑月娥等人</a><br />
英国外交部官员首次表示，英国要实施新的法律，来制裁在镇压香港抗议活动中侵犯人权的嫌犯。跨党派香港议会小组（APPG）的秘书怀特豪斯（Chris Whitehouse）表示，香港特首林郑月娥及一些香港高级警官显然是该法律制裁的目标。</p>
<p>8. <a href="https://github.com/fwwipt2046/djy/blob/master/gb/19/11/14/n11655457.md">冤狱酷刑 颠沛流离 法轮功学员邓莉的遭遇</a><br />
新疆乌鲁木齐市法轮功学员邓莉多次被非法拘禁、劳教、判刑，遭受酷刑折磨。因不堪中共人员没完没了的骚扰迫害，被迫离家出走，回到原籍河南，后被新疆公安列为“恐怖分子”追捕，至今一直颠沛流离。</p>
<p>9. <a href="https://github.com/fwwipt2046/djy/blob/master/gb/19/11/14/n11655166.md">程晓容：香港――留给谁的时间不多了？</a><br />
共产党的统治伴随着无尽的灾难，时至今日，中共仍未停止谋杀――杀大陆人、杀香港人，残杀心怀正气的好人，扑杀抗拒邪恶的精神。拥护这个党，坚持这个党的领导，强迫军队、媒体、司法和所有人民跟党走，就是在犯罪，必将受到清算和惩罚。</p>
<p>10. <a href="https://github.com/fwwipt2046/djy/blob/master/gb/19/11/13/n11653410.md">近期四川法轮功学员遭迫害概况</a><br />

<p>11. <a href="https://github.com/fwwipt2046/djy/blob/master/gb/19/11/14/n11656410.md">北京查公款游 专家：国库空虚 中共设法搜刮</a><br />

<p>12. <a href="https://github.com/fwwipt2046/djy/blob/master/gb/19/11/14/n11655530.md">数百陆生上传证件撑港“自由不死中共必亡”</a><br />

<p>13. <a href="https://github.com/fwwipt2046/djy/blob/master/gb/19/11/14/n11656778.md">负债累累 今年以来逾4百家房地产企业破产</a><br />

<p>14. <a href="https://github.com/fwwipt2046/djy/blob/master/gb/19/11/14/n11655590.md">国会报告：若中共镇压 美取消香港特惠待遇</a><br />
